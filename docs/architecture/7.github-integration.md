# GitHub Integration

> Source code tells us what exists.
>
> GitHub tells us why it exists.

---

# Why GitHub Matters

A repository only captures the current state of a project.

It doesn't explain:

- Why a feature was added
- Why an API changed
- Which discussion led to an implementation
- Which issue introduced a bug
- Who reviewed the change

Those answers live inside GitHub.

Cliper extends repository intelligence by learning from the entire development lifecycle.

---

# Real World Analogy

Imagine walking into a completed house.

You can see:

- the walls
- the kitchen
- the bedrooms

But you don't know:

- why the kitchen was moved
- why a wall disappeared
- why another room was added

To understand those decisions, you need the architect's notes.

GitHub is the architect's notebook for software.
